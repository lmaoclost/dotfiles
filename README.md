# DotFiles WIP
A simple shell script used to configure my desktop. Coded by [juniorcl](https://github.com/juniorcl) that i adjusted to my use.

## About
This is a shell script created for configuration my desktop. This can be used for different operating systems (OSs) based on GNU/Linux I commonlly use. The list below shows them.

#### Supported OSs
- ArchLinux
- Ubuntu (coming soon)

## Usage
Create a folder in on your `$HOME`, clone the repository inside it and execute the [install.sh](install.sh). You can execute the commands below to run the script.

```
$ chmod +x install.sh && ./install.sh
$ ./install.sh
```

A menu will show up where you can select the operating system for setting up.
